# Code of Conduct

## Our Pledge

We want Mimir to be a welcoming project for contributors of all experience levels. We commit to making participation respectful, constructive, and harassment-free.

## Our Standards

Examples of behavior that help create a positive environment:

- Being respectful in language and tone
- Giving actionable, kind feedback
- Accepting good-faith disagreement
- Prioritizing shared understanding over ego
- Helping newer contributors learn the codebase

Examples of unacceptable behavior:

- Harassment or discrimination
- Personal attacks
- Trolling or deliberately hostile behavior
- Publishing private information without permission
- Repeatedly derailing technical discussions

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct. They may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate these standards.

## Reporting

If you experience or witness unacceptable behavior, open a private report to the maintainer at `eminuckan@gmail.com`.

## Scope

This code of conduct applies within project spaces and in public spaces when someone is representing the project.
